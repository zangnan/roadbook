使用说明

了解如何使用路书的三大核心功能

照片轨迹

从照片提取 GPS 信息，自动生成交互式轨迹地图和时间轴

Web 使用步骤

  1. 将包含 GPS 信息的照片放入 photo/<目录名>/ 目录
  2. 访问 照片轨迹 页面
  3. 选择或输入目录名
  4. （可选）调整距离阈值和时间阈值参数
  5. 点击「开始生成」，等待处理完成
  6. 点击「查看轨迹地图」或「查看时间轴」浏览结果

CLI 使用方式

python src/photo_track.py <照片目录> [选项]
-H, --html-only跳过图片解析，仅生成 HTML
-d, --distance-threshold距离阈值（米），默认 1000
-t, --time-threshold时间阈值（秒），默认 7200
-v, --verbose详细日志输出

路径规划

基于高德地图的路线规划功能，支持多种出行方式

使用步骤

  1. 访问 路径规划 页面
  2. 在搜索框中输入起点，按回车添加
  3. 继续添加途经点和终点（最多支持 16 个途经点）
  4. 点击「开始规划」，查看路线详情

AI路书

AI 生成完整旅行行程，包含景点、住宿、餐饮推荐和预算估算

使用步骤

  1. 访问 AI路书 页面
  2. 点击页头右上角AI生成按钮
  3. 填写行程标题、天数、人数、旅行日期范围
  4. 选择人数和房间数
  5. 选择车型（轿车/SUV/越野车/MVP）
  6. （可选）填写预算偏好和特殊需求
  7. 点击「生成路书」，等待 AI 完成
  8. 查看生成的行程，可收藏、导出Json、导出为 Excel

配置 API Key

AI 路书功能需要配置 DeepSeek API Key，请在 .env 文件中设置：

DEEPSEEK_API_KEY=your_api_key_here
DEEPSEEK_API_URL=https://api.deepseek.com/v1
DEEPSEEK_MODEL=deepseek-reasoner

常见问题

如何获取高德地图 API Key？

1. 访问 高德开放平台控制台

2. 注册账号并完成开发者认证

3. 创建应用，获取 Web API Key 和 Web Service Key

4. 在 .env 文件中配置：

AMAP_WEB_AK=你的Web端Key
AMAP_SERVER_AK=你的服务端Key
照片没有 GPS 信息怎么办？

路书依赖照片中的 EXIF GPS 信息来生成轨迹。如果照片没有 GPS 信息：

  • 可以手动添加 GPS 坐标（需要修改 EXIF 数据）
  • 或使用「仅生成 HTML」模式，手动编辑生成的 JSON 数据

提示：手机拍摄的照片通常包含 GPS 信息，但经过压缩或编辑的照片可能会丢失。

如何配置 DeepSeek API Key？

1. 访问 DeepSeek 开放平台

2. 注册账号并获取 API Key

3. 在项目根目录的 .env 文件中添加：

DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx

4. 重启应用即可使用 AI 路书功能

生成的轨迹地图如何分享？

生成的轨迹地图是独立的 HTML 文件，可以：

  • 直接打开 output/<目录名>/track_output.html 文件
  • 将整个输出目录压缩打包分享给其他人
  • 部署到 Web 服务器上访问

注意：地图需要联网加载高德地图资源，离线无法正常显示。

桌面应用和 Web 应用有什么区别？

桌面应用（PyWebView）：通过 build.bat 打包的独立可执行文件，适合不熟悉命令行的用户。

Web 应用：通过 python src/web_app.py 启动，适合开发者或需要远程访问的场景。

两者功能完全一致，界面相同。

距离阈值和时间阈值是什么？

这两个参数用于控制轨迹点的「合并策略」：

  • 距离阈值（默认 1000 米）：相邻两点距离小于此值时，可能被合并
  • 时间阈值（默认 7200 秒 = 2 小时）：相邻两点时间差小于此值时，可能被合并

如果同一天拍的照片间隔很近，系统会将其合并为同一个地点节点，避免轨迹过于密集。
